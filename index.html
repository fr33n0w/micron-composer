Micron Syntax Reference

Text Formatting

`!Bold text`! - Bold text
`*Italic text`* - Italic text
`_Underlined text`_ - Underlined text
`` - Reset all formatting

Colors

`FxxxText`f - Foreground (text) color (xxx = 3-digit hex)
`BxxxText`b - Background color (xxx = 3-digit hex)
`Ff00Red text`f - Example: Red text
`B0f0Green background`b - Example: Green background
`Ff00`B0f0Colored text`b`f - Both foreground and background

Text Alignment

`cCentered text`a - Center align
`lLeft aligned text`a - Left align
`rRight aligned text`a - Right align
`a - Reset to default alignment

Headers

>Header Level 1 - Main header
>>Header Level 2 - Subheader
>>>Header Level 3 - Sub-subheader
< - Reset section depth

Links

`[Link text`/path/to/page.mu]` - Basic link
`[`destination.mu]` - Link without custom text
`!`[Bold link`page.mu]`!` - Bold link
`_`[Underlined link`page.mu]`_` - Underlined link
`[Link`page.mu`field1|field2]` - Link with form fields

Input Fields

`<fieldname`default value> - Basic text field
`<24|fieldname`default value> - Field with custom width (24 chars)
`<!16|password`Enter password> - Masked password field
`<?|checkbox|value`>Label text - Checkbox
`<?|checkbox|value|*`>Checked by default - Pre-checked checkbox
`<^|groupname|value`>Option 1 - Radio button
`<^|groupname|value|*`>Selected option - Pre-selected radio

Horizontal Dividers

- - Standard horizontal line
-= - Custom divider (repeats character after dash)
-━ - Example: Heavy line divider
-★ - Example: Star divider

Literal Text Mode

`= - Toggle literal mode (display markup as-is)
\`= - Escape literal toggle to display "`="

Comments

# This is a comment - Comments are not displayed

Special Characters

\` - Escape backtick to display it literally
\\ - Escape backslash

Combining Formatting

`!`*Bold and italic`*`! - Multiple formats
`Ff00`!Red bold text`!`f - Color + bold
`c`Ff00`!Centered red bold`!`f`a - Alignment + color + bold

Color Examples

fff - White
000 - Black
f00 - Red
0f0 - Green
00f - Blue
ff0 - Yellow
f0f - Magenta
0ff - Cyan
f80 - Orange
58a - Light blue

Tips

• Keep lines under 130 characters for MeshChat compatibility
• Use `` to reset all formatting at once
• Formatting codes can be nested but must be properly closed
• Field names should be unique within a page
• Radio buttons with the same group name are mutually exclusive
